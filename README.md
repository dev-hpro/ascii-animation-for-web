# ascii-animation-for-web
Projeto para transformar frames de imagem de qualquer formato em Animação ASCII que roda direto do navegador.


***

Instrução de uso:
Transforme o vídeo que você quer em PNG,m JPG ou qualquer outro formado de imagem, use algum programa para extrair os frames exemplo kdenlive
copie todos os frames em sequencia para pasta input 
depois npm install no dir
em seguida node servidor.js

abre localhost:8000
e você tera uma acesso a interface, ao final do projeto que você criar você pode exportar como HTML puro 

OBS: quanto mais frames, maior fica o arquivo, ideal para pequenas animações e transições
é importante lembrar que mesmo que rode no seu computador verifique o consumo de ram do navegdor na hora da build tem que ter bastante memoria

***

Este é um projeto para criação de Animações usando ASCII voltados para quem quer implementar em um APP ou coisa do tipo.
Quanto mais char e frame tiver maior será o peso, idel para criar pequenas animações e inserir em paginas dando um deferencia baaca.
